# GameRules Reference

Complete reference for all GameRules registered by **IG: Magnet, Let me get that!**.

## Configuration

| Rule | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `ig_magnet_range` | Integer | 12 | The radius (in blocks) within which items and XP orbs are pulled towards the player. |
| `ig_magnet_noclip` | Boolean | true | If true, items and XP orbs will be pulled through solid blocks. |
| `ig_magnet_affects_xp` | Boolean | true | If true, the magnet will also pull XP orbs. |
| `ig_magnet_particles` | Boolean | true | If true, visual particle effects will be displayed when items are pulled. |
| `ig_magnet_particle_count` | Integer | 1 | The number of particles spawned per tick per entity when pulled. |
| `ig_magnet_speed` | Integer | 80 | The pulling speed as a percentage of maximum velocity (1-100). |
| `ig_magnet_acceleration` | Integer | 10 | The acceleration rate as a percentage (1-100). |

## Category

All rules appear under the **Miscellaneous** category in the vanilla *Edit Game Rules* screen.
